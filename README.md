# 简介 

自己写一些Echart小项目, 方便以后拿来就用

# 项目

## provinces

* 文件说明

| 文件| 说明
|--|--|
|geoinfo.json | 中国县及以上行政区的经纬度数据集
|changjiang.json| 长江的多边形经纬度点
|allpos_t.json | 企业迁移数据集
|geo_tool.js | 位置处理工具，主要是把地名转为经纬度
|tool.js| 通用工具

* 使用说明

下载打开index.html即可
